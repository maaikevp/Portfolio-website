Meet app

Meet is an app to locate great events This app uses both serverless architecture and Open authorization (with Google). A serverless, progressive web application (PWA) is built with React using a test-driven development (TDD) technique. The application uses the Google Calendar API to fetch upcoming events, and Google OAuth is used for verification. The serverless function is implemented using AWS Lambda. Systems and programs used:

  • Google API and Google OAuth
  • AWS Lambda
  • React.js and TDD techniques
  • HTML, CSS
